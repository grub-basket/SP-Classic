# 0.95.4 — Folder panel placement, native settings search, panel polish

## Folder panel placement (0.95.0–0.95.2)
Take control of the folder list and the Pinned shelf.

- **Pin / downrank / hide folders.** Right-click any folder in the panel to
  **pin** it to the top (clustered with a pin icon), **downrank** it into a
  dimmed group at the bottom, or **hide** it from the list. Each folder is in at
  most one state.
- **Restore anywhere.** Hidden folders collapse into a **"Hidden (N)"** strip at
  the bottom of the list with an inline unhide, and are also listed in
  **Settings → General → "Folder panel placement"** (Unpin / Reset / Unhide).
- **Reload-proof open indicators.** The per-folder "open" dot now lights up for
  every folder with an open tab — including tabs Obsidian lazy-defers after a
  restart — by reading each tab's persisted folder instead of only the focused
  one. Exact immediately on launch, no caching.
- **Folder icons.** Each folder row gets an icon, tinted by that Stashpad's Home
  note color when one is set.

## Pinned shelf, redesigned (0.95.1–0.95.4)
The folder panel's Pinned list now matches the Pinned side panel.

- **Expandable subtrees.** Parent pins show a child-count badge; click to expand
  the outline inline.
- **View options.** A menu (list icon) toggles **Sort by pin order** (default) or
  **Group by folder** — shared across both the folder panel and the Pinned panel.
- **Drag to reorder** pins in the folder panel.
- **Stronger note colors** carry into both pin panels: a left accent bar, a tinted
  wash, and a tinted icon + label.
- The Pinned panel's view-options button is tucked onto the Home row.

## Native settings search (0.94.0–0.94.4)
Stashpad's settings migrated to Obsidian 1.13's declarative settings API.

- Stashpad settings now appear in Obsidian's built-in **"Search settings…"**.
- **Per-setting search** for General, Diagnostics, and every Hotkey command —
  search "command palette", "timezone", "confirm deletes", etc. and jump straight
  to the control (~80 individually searchable entries).
- Settings render as native sub-pages; the old custom tab bar and in-plugin search
  box are gone. `minAppVersion` is now **1.13.0**.

## Focus + navigation (0.95.3)
- **Focus last Stashpad tab** and **Focus folder panel** commands bounce focus
  between the side panels and your notes — layout-independent, bindable from
  Obsidian's Hotkeys.

## Folder deletes are safer (0.95.1–0.95.2)
- Deleting a folder from the panel **closes its open tab(s)**, posts a persistent
  notification, and offers **Undo** (the folder is moved to the vault trash and
  can be restored in one click). Deletions from the file explorer notify too.

## Fixes & polish (0.92.x–0.93.x)
- **iPad / mobile icons** no longer collapse to zero width (blank nav and search
  buttons), with a general fix to keep inline icons from being squeezed.
- **File-explorer context menu:** "Open folder in Stashpad" on existing Stashpad
  folders.
- **Hotkeys:** a revert-to-default icon on each binding slot.
- **Search scope:** an "excluded Stashpad folders" opt-in at the bottom of the
  folder picker.
- Keep a multi-selection when pressing Escape out of the composer.
- Auto-detect synced/deployed builds and nudge for a reload without needing a
  window refocus.
