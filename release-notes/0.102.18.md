# Stashpad Classic 0.102.18

Feature back-port: pulls several improvements from mainline Stashpad into Classic —
all using only stable-Obsidian APIs (no insider-only APIs, no encryption).

## New

- **Deep links.** Copy an `obsidian://stashpad?…` link to any note (command **Copy
  Stashpad link**, or the note's right-click menu). Click it from anywhere — another
  app, a daily note, a browser — to jump straight back to that note. Links use the
  note's stable id, so they survive renames.

## Improved

- **Smarter search results.** The note picker now ranks matches by relevance (exact
  title → title prefix → phrase → tokens → body) and then by most-recently-edited,
  instead of raw tree order. An empty query lists your most recent notes first.
- **Breadcrumbs show real titles.** A note with no heading now shows its first body
  line in the breadcrumb and tab title, instead of its filename slug.
- **Hotkeys fire more reliably.** Shift+digit and (macOS) Alt/Option+letter bindings
  now register correctly, and single-key symbol bindings like `&` (merge) work.
- **Default "Toggle task" hotkey is now G** (was H); existing bindings migrate once
  and any rebind you make afterward sticks.

## Fixed

- **Note IDs never collide.** New notes are now guaranteed a unique id at creation
  (checked against every id in the vault), eliminating the small-but-real chance of
  a duplicate id in large vaults.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.

> ⚠️ **Alpha software, AI-built.** Not human-audited. Keep backups. Please report
