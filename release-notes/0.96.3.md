# 0.96.3 — Fix plugin failing to load on older Obsidian

Follow-up to 0.96.2. That release lowered the minimum Obsidian version but the
plugin still failed to load on pre-1.13 Obsidian: a settings class extended a
1.13-only API at module load, which threw before the plugin could start.

- The plugin now loads correctly on **pre-1.13 Obsidian**, with a working
  (non-blank) settings tab. On Obsidian 1.13+, settings and native search are
  unchanged.

If 0.96.2 wouldn't load for you on an older Obsidian, this fixes it.
