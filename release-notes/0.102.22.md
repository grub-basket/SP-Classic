# Stashpad Classic 0.102.22

A rendering bugfix + cleanup.

## Fixed

- **No more stuck "earlier version" of a note.** The body-render cache is keyed by
  modified-time; on a network drive or after an external/synced edit, a render could
  stamp the new time onto old content and then keep showing a truncated,
  attachment-less earlier version until you reloaded. Edited notes now drop that cache
  entry so the view recomputes from fresh content.
- Removed a leftover debug log that printed on every search.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.
