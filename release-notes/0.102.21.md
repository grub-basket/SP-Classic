# Stashpad Classic 0.102.21

Folder-panel improvements + a review-lint fix.

## New

- **Pinned notes in the folder switcher.** New setting (off by default) — when on,
  the folder switcher / creator also lists your pinned notes so you can jump straight
  to one, not just to folders.
- **Reveal a folder's pinned notes in the panel.** Each folder row in the folder
  panel now shows a chevron + count when the folder has Home (top-level) list-pinned
  notes; click to expand an inline list and jump to any of them.

## Fixed

- Internal: a style was set directly instead of via `setCssStyles` (community-review
  lint) — no visible change.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.
