# 0.51.11 — First public release

Stashpad is a chat-style nested-notes view for Obsidian: rapid capture,
outliner navigation, in-place editing. Local-only, no external services.

This release captures everything from the project's genesis through 0.51.11.

---

## The composer + capture loop
- Chat-style composer with autofocus-after-send (setting).
- Per-folder drafts that survive reload, tab restore, and folder switches;
  cross-tab draft sync; debounced per-keystroke saves so live typing isn't
  clobbered.
- Composer autocomplete for tags (`#`) and wikilinks (`[[`) with
  order-independent token matching; focus and caret preserved across renders.
- Drag-and-drop or paste files into the composer (0.48).
- Split a note at the cursor line via modal (0.37); the second half inherits
  the source's creation timestamp `+1ms`.

## The outliner / list
- Note reordering via keyboard (0.16) and full drag-and-drop with hover grip,
  cross-parent drag (with confirm), drop-to-nest, animated drop indicator,
  toasts on reparent (0.17).
- Outdent (`Mod+[`) re-parents the selection to its grandparent (0.23).
- Arrow-wrap at list edges; click a link to open the target; tab focus
  returns to the originating Stashpad when an Edit-in-Tab closes (0.19, 0.29).
- Copy-as-outline of a selection (0.19).
- Per-note color coding: swatch + border + child arrow; preset grid +
  custom-color submenu; per-Stashpad color aliases (0.24, 0.46).
- "Adopt" outside notes into a Stashpad via the command palette; auto-rename
  to `<basename>-<id>.md`; auto-fix orphan parents on create / rename into a
  Stashpad (0.22, 0.34).
- Cross-Stashpad search and move (0.32); cross-Stashpad results tier below
  locals in the search modal (0.31).

## Undo / redo
- Undo/redo for note creation (with text restored to the input box),
  deletion (immediate), draft deletion, reorder, reparent, color, and more.

## Filters + search
- Time filter (Today / Week / Month / Year / All) with a calendar/rolling
  toggle, mode-aware labels, persisted across reloads (0.22, 0.42).
- Tag filter dropdown — subtree-scoped, immediate-children counts, clears on
  navigation (0.41).
- Color filter dropdown with per-Stashpad aliases (0.45, 0.46).
- Search/pick rows show body on top with parent body below (0.30).
- In-list picker preserves scroll, pre-selects above the cursor; Esc
  cancels and keeps scroll pinned at bottom (0.20, 0.21).

## Multiplayer authorship (the 0.51.x headline)
- Per-user identity in settings: name, role, department.
- `_authors/` reserved folder with an auto-generated stub file per identity;
  bidirectional sync between the settings name and the stub.
- Author stamped on every new note; contributor list tracks who else has
  edited it (debounced until edits settle).
- Footer row on each note: author / contributors / last-edit, with
  per-folder "you've worked in" links.
- Log gains an Author column so multi-user activity is auditable.

## Keybindings + commands
- Unified per-command bindings: two slots per command + active-slot toggle +
  in-app recorder (0.26).
- Keybind/command parity audit (0.50): every Stashpad command is now
  bindable from the command palette.
- Clone-the-current-tab command (0.51.11).
- `Mod` hint shown as `(Cmd/Ctrl)` for cross-platform clarity.

## Mobile
- A multi-pass mobile redesign (0.36): lift composer above the keyboard,
  hide Obsidian's mobile toolbar over Stashpad, floating composer rail,
  select-mode toggle + `⋯` actions menu, focused-header sticky one-line
  preview when scrolled offscreen, focus-driven keyboard-lift via
  `:has(textarea:focus)`. Desktop also gets the `⋯` actions menu (0.38).

## Layout / chrome
- Focused header redesigned to match list-row layout (3 columns +
  duplicate-tab button) (0.40).
- Multi-select with Esc collapsing to the anchor; zap icon select-mode
  toggle; multi-edit (0.38, 0.39).
- Tag click opens global search filtered by that tag (0.19).
- Breadcrumb context menu, "open in tab" actions (0.37).

## Filesystem + integrity
- Plugin private state relocated into `<pluginDir>/.stashpad/` (0.43).
- Folder-scoped integrity sweep, paginated log with export + clear (0.21).
- `data.json` writes serialized so draft writes can't clobber shortcuts.
- Confirm-delete settings split for multi-delete and single notes with
  descendants/attachments (0.44).
- Stricter Stashpad detection + create-Stashpad UI in settings (0.35).

## Templates + cloning
- Clone, insert template, per-Stashpad template file; warns when template
  frontmatter overlaps Stashpad's auto fields (0.47).

## Sticky scroll-to-bottom
- Sticky scroll across async markdown rendering; render restores to the NEW
  bottom when previously bottom-pinned (0.20, 0.21).

## Misc polish
- Esc consumed via Obsidian's `Scope` so it never warps tabs (0.49).
- `:has()`-free where the perf matters; rendered-markdown cache per row
  (later in 0.52 series — included here for context).

## Install
Compatible with Obsidian 1.7+. Install via BRAT from this repo, or copy
`main.js`, `manifest.json`, and `styles.css` into a folder under your vault's
`.obsidian/plugins/`.
