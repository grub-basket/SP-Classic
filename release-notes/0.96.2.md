# 0.96.2 — Runs on older Obsidian again; compact-mode checkboxes

## Compatibility: runs on pre-1.13 Obsidian again (0.96.2)
The settings overhaul in 0.94.0 had raised the minimum Obsidian version to 1.13.0
(for the native settings-search integration), which left the plugin unusable on
older releases. That's fixed:
- **`minAppVersion` is back to 1.7.0.** The plugin loads and the settings tab
  works on older Obsidian again.
- On Obsidian **1.13+** you still get the native settings-search integration. On
  **older** versions the settings render normally (just without the in-app
  settings search, which those versions don't have anyway).

## Compact mode: checkboxes on every row (0.96.1)
- In **compact mode**, every note row now shows a checkbox so the list reads as a
  tight checklist — not only task-tagged notes. Click a checkbox to toggle that
  note's done state in place.
