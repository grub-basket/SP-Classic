# 0.96.0 — Search opens in a new tab; second color-picker chord

## Search results open in a new tab (0.96.0)
- New setting **"Search results open in a new tab"** (Settings → General, **on by
  default**). Picking a result in the Search modal now opens it in a fresh
  Stashpad tab focused on that note, instead of navigating the current tab.
- Applies to both same-folder and cross-Stashpad results. Turn it off to restore
  the previous in-place navigation.

## Set note color — second default chord (0.95.5)
- The **Set note color** command now defaults to both **`Shift+:`** and **`;`**
  (semicolon), with both chords active. Existing users who hadn't remapped it
  pick up the semicolon automatically; anyone who customized it is left alone.
