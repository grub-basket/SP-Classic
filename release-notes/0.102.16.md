# Stashpad Classic 0.102.16

**The first release of Stashpad Classic** — a fork of Stashpad for **stable
(non-insider) Obsidian**.

Mainline Stashpad raised its `minAppVersion` to `1.13.0` (insider builds only) from
0.102.19 onward. Stashpad Classic is pinned at **0.102.16**, the last release that
still targets stable Obsidian (`minAppVersion 1.7.0`), so you can run it on a normal,
publicly released Obsidian.

## What's different from mainline 0.102.16

- **Encryption removed.** All encryption features are disabled in Classic — the
  lock/unlock commands, the Encryption settings tab, the folder and note
  context-menu options, archive-folder auto-encryption, and the encrypted trash.
  Encryption was the feature most tied to insider-only APIs, so Classic drops it.
- Everything else is the full Stashpad 0.102.16 feature set: chat-style nested notes,
  outliner navigation, fast search, tasks, per-folder templates, and one-click
  **Open Knowledge Format (OKF)** export.

## Install

Via [BRAT](https://github.com/TfTHacker/obsidian42-brat): add the beta plugin
`grub-basket/SP-Classic`. Or download `main.js`, `manifest.json`, and `styles.css`
from this release into `.obsidian/plugins/stashpad-classic/` in your vault.

> ⚠️ **Alpha software, AI-built.** Stashpad does a lot and is young; the code was
> written by an AI assistant and is not human-audited. Keep backups. This is the
> first Classic build — please report anything that misbehaves on stable Obsidian.
