# Stashpad Classic 0.102.24

Performance + navigation back-ports (stable-API only, no encryption).

## Improved

- **Instant note creation — no flash.** Creating a note used to briefly show the
  filename, then re-read the body over the network before showing it (a visible
  body→title→body flash on network drives / sync). New notes now paint their real
  body immediately, and the follow-up recovery-field write no longer triggers a
  second flash.
- **Jump to any breadcrumb level.** A new button at the start of the breadcrumb row
  (always visible, even when the path is deep and the crumbs are clipped) opens a
  list of every level in the path — full titles, no truncation — so you can jump
  straight to any ancestor.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.
