# Stashpad Classic 0.102.23

More back-ported features from mainline (all stable-API, no encryption).

## New

- **Per-folder tab icons.** Right-click a folder in the folder panel → **Set tab
  icon…** and enter a [Lucide](https://lucide.dev) icon id (e.g. `rocket`, `star`).
  The folder's tab, switcher, and panel row show it instead of the default.
- **Copy text / Cut note in the right-click menu.** Both now appear on every note's
  menu, and the focused-note header opens that same menu.
- **Deep-link polish:** a copy-link button on each note row (desktop); **Copy Stashpad
  link** with several notes selected now copies one link per note; and a new **Open
  Stashpad link** command lets you paste a deep link to jump to it (for apps that
  won't render `obsidian://` as clickable).

## Fixed

- **List timestamps honor your date format.** The two-line timestamp in the notes
  list ignored the "Date display format" setting — now it follows the dropdown
  (and timezone), matching the detail panel.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.
