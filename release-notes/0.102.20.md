# Stashpad Classic 0.102.20

Sync fix + panel/navigation improvements.

## Fixed

- **Settings now sync across devices.** Folder-panel pins/hides, toggles, and
  keybindings weren't syncing — Classic stored its `data.json` in a private folder
  that Obsidian Sync doesn't cover (a leftover of the removed encryption design).
  Settings now live at the standard, synced path; your existing settings migrate
  automatically on first launch (a `.bak` is kept for safety).
- **Search results return you to where you searched from.** Closing a note you
  opened from search now focuses the tab you searched from, instead of the tab to
  its right. (The other open-in-new-tab paths already did this.)

## New

- **Create a Stashpad folder from the folder panel.** A folder-plus button in the
  Folders heading opens a quick name prompt, creates the folder (seeded with a Home
  note), and opens it — no trip to Settings.

## Compatibility

`minAppVersion` stays at **1.7.0** — installs on the current stable Obsidian desktop
release and older.

> ⚠️ **Alpha software, AI-built.** Not human-audited. Keep backups.

> Note: the first cross-device launch after this update does a one-time
> last-writer-wins on settings (the migrating device's `data.json` wins); after that,
> sync is normal.
